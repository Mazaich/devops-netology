2.Благодаря добавленному файлу .gitignore в каталоге terraform/, система контроля версий Git будет игнорировать следующие типы файлов и каталогов,
 соответствующие описанным в нем шаблонам:

.terraform/ — Будет проигнорирована папка с именем .terraform, а также всё её содержимое (файлы и вложенные папки), где бы она ни находилась в проекте.

*.tfstate — Будут проигнорированы все файлы, имя которых оканчивается на .tfstate.

*.tfstate.* — Будут проигнорированы все файлы, у которых в названии есть часть .tfstate., а после неё следует любое продолжение.

crash.log — Будет проигнорирован конкретный файл с именем crash.log.

crash.*.log — Будут проигнорированы файлы, имена которых начинаются с crash., 
затем содержат любой текст, и заканчиваются на .log.

*.tfvars — Будут проигнорированы все файлы с расширением .tfvars.

*.tfvars.json — Будут проигнорированы все файлы с расширением .tfvars.json.

override.tf и override.tf.json — Будут проигнорированы конкретные файлы с такими именами.

*_override.tf и *_override.tf.json — Будут проигнорированы файлы, имена которых заканчиваются на _override.tf или _override.tf.json
Символ * в начале шаблона означает любое количество любых символов перед _override.

.terraform.tfstate.lock.info — Будет проигнорирован конкретный файл с таким именем.

.terraformrc и terraform.rc — Будут проигнорированы конкретные файлы с такими именами.
